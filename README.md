# QA Testing Demo

Personal QA project demonstrating a complete testing approach on an e-commerce context.

## Scope
- API testing using Fake Store API
- End-to-end testing using Playwright
- Manual testing and documentation
- QA metrics and test strategy

## Tools
- Playwright
- Bruno (API testing)
- GitHub
- Jira (bug tracking & KPI)

## Goal
Showcase testing skills across the testing pyramid: API, E2E, and quality monitoring.
